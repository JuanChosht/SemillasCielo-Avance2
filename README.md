# Semillas del Cielo - Avance 2

Juego de plataformas 2D desarrollado en Unity como parte del curso de Desarrollo de Videojuegos.

## Descripcion

Controla a Corvus, un cuervo que debe recolectar semillas y sobrevivir en un mundo hostil. Esquiva enemigos, recoge proyectiles y avanza por los niveles.

## Instalacion y ejecucion en Unity

### Requisitos
- Unity 2022.3 LTS o superior
- Universal Render Pipeline (URP) incluido en el proyecto

### Pasos
1. Clona el repositorio:
   ```
   git clone https://github.com/JuanChosht/SemillasCielo-Avance-2.git
   ```
2. Abre Unity Hub.
3. Haz clic en **Add** y selecciona la carpeta clonada.
4. Abre el proyecto desde Unity Hub (Unity descargara los paquetes automaticamente).
5. Ve a `File > Open Scene` y abre `Assets/Scenes/SampleScene.unity`.
6. Presiona **Play** para probar el juego.
